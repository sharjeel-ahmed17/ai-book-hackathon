# Physical AI Humanoid Robotics Textbook

Welcome to the comprehensive guide to Physical AI and Humanoid Robotics. This textbook supports a 13-week training program for industry engineers with Python knowledge, covering essential topics from foundational concepts through advanced humanoid robotics applications.

## Course Structure

The curriculum is organized into 5 main sections plus a capstone project:

- **Weeks 1-2**: Foundations of Physical AI
- **Weeks 3-5**: ROS 2
- **Weeks 6-7**: Digital Twin workflows
- **Weeks 8-10**: NVIDIA Isaac
- **Weeks 11-13**: VLA models and humanoid robotics
- **Capstone**: Build an autonomous humanoid pipeline

## Learning Objectives

By the end of this course, you will be able to:
- Understand the fundamentals of Physical AI and its applications
- Work with ROS 2 for robotics development
- Create and use digital twin environments
- Leverage NVIDIA Isaac tools for robotics
- Implement VLA models for humanoid robotics
- Build an end-to-end autonomous humanoid pipeline
